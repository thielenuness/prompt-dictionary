# Prompt Dictionary

An interactive prompt dictionary generated with Claude Code.

Live demo: https://thielenuness.github.io/prompt-dictionary/

---

## About

This project is a simple HTML-based prompt dictionary designed to help developers and creators explore, understand, and reuse prompt patterns.

It was generated using AI (Claude Code) and published with GitHub Pages.

---

## Features

- Clean and minimal interface  
- Easy navigation  
- Lightweight (pure HTML)  
- Hosted on GitHub Pages  

---

## Tech Stack

- HTML  
- GitHub Pages  

---

## Usage

1. Open the live demo  
2. Browse the prompts  
3. Copy and reuse as needed
